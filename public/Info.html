NASA Image Search Application

✅What is NASAIMG?

NASA API를 이용한 NASA image 검색 application입니다. search 페이지로 이동하여 keyword 검색시, NASA API를 통해 검색된 이미지들이 카드형태로 제공됩니다.

✅MUI

해당 프로젝트는 Material-UI(MUI, 머티리얼 디자인)을 적용해 진행하였습니다. MUI는 여러 스타일 가이드 중 하나로, 구글에서 공개한 디자인 가이드입니다.

MUI는 컴포넌트 단위 개발을 할 때 굉장히 유용하며 완성도 있는 디자인을 제공합니다. 하지만 그만큼 커스터마이징이 어려우며, Bootstrap과 비교했을 때 MUI는 대부분 UI 구성 요소의 집합이기때문에 Bootstrap만큼 개발속도를 향상시키지 못합니다. 또한 Bootstrap에 비해 커뮤니티 규모가 작습니다.

하지만 모바일 친화적이며 빠른 시간 내에 완성도 있는 컴포넌트 개발을 할 수 있다는 점에서 효과적인 CSS 프레임워크인 것 같습니다.
